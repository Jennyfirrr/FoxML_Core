### GPU Acceleration
- **Status**: ‚úÖ Enabled for target ranking and feature selection
- **Supported Models**:
  - LightGBM: CUDA and OpenCL support
  - XGBoost: CUDA support (new and legacy API)
  - CatBoost: GPU support
- **Configuration**: All settings in `CONFIG/training_config/gpu_config.yaml` (SST)
- **Recent Updates** (2025-12-12):
  - Fully config-driven (no hardcoded values) - SST compliance achieved
  - All GPU detection parameters use `get_cfg()` directly (not `.get()` with hardcoded defaults)
  - Automatic GPU detection with graceful fallback
  - Support for disabling GPU test for faster startup (`test_enabled: false`)
  - Better logging: Always shows GPU status (not just when `gpu_detail` enabled)
  - XGBoost: Tries both new API (`device='cuda'`) and legacy API (`tree_method='gpu_hist'`)

- **Recent Updates** (2025-12-12):
